# 私有基准样本：杜拉斯《情人》开头 / 王道乾译本

## 用途

此样本用于检验 AI 译文是否具备文学翻译的中文节奏、叙述克制和审美强度。

它不是要让待翻书模仿杜拉斯，也不是要把探险纪实翻成抒情小说；它的作用是建立一把标尺：优秀译文不是逐句搬运，而是在忠实原意的前提下，让中文本身成立。

## 这个开头片段要学习什么

### 1. 叙述声音

- 第一人称不是说明事实，而是在回望自身。
- 语气冷静、笃定、带有距离感。
- 情绪不靠形容词堆叠，而靠句子推进自然显现。

### 2. 中文节奏

- 短句负责定调，长句负责铺展。
- 重复不是冗余，而是回环和加深。
- 长句内部有停顿、有层次，不是一串从句硬接。

### 3. 意象处理

- “形象”“面貌”“衰老”等词反复出现，但每次功能不同。
- 抽象判断要落在可见的面容、眼神、皱纹、皮肤变化上。
- 译文允许重排语序，以保护中文的自然推进。

### 4. 审美控制

- 不把句子翻得过分华丽。
- 不滥用成语。
- 不把心理描写翻成解释性白话。
- 关键句要有余味，而不是只完成信息传递。

## 对待翻书的迁移原则

用于《黑人北极探险家》这类探险纪实时，不迁移其抒情腔，而迁移以下能力：

- 开篇句要有定调能力。
- 回忆性段落要有口述者的生命感。
- 探险段落要有现场压力和空间感。
- 历史判断句要庄重、清楚，避免直译腔。
- 重要句子宁可重译三次，也不要保留“看得懂但不好读”的版本。

## PASS 标准

用此样本测试翻译方法时，AI 自译文本需要达到：

- 语义不误读。
- 中文读起来不像英文句法换皮。
- 重复有节奏，不显啰嗦。
- 情绪有克制，不滥情。
- 至少有一版译文能说明：为什么这样处理比直译更好。
