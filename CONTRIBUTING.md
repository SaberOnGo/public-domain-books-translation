# Contributing / 贡献说明

Thank you for helping improve public-domain book translations. By submitting a pull request, issue suggestion, translation, edit, review, glossary entry, metadata change, cover/layout improvement, or any other contribution to this project, you agree to the rules below.

## 简体中文

### 贡献内容

贡献可以包括但不限于：

- 翻译、改译、补译。
- 审校、润色、错字修复。
- 术语表、风格指南、译注、metadata。
- EPUB 排版、封面、脚本和自动化流程。
- 问题反馈、试读意见和质量检查记录。

### 公开发布

你提交的非代码贡献可以被纳入本项目，并按 [LICENSE.md](LICENSE.md) 中的 `CC BY-NC-SA 4.0` 公开发布。

### LifeBook 书坊使用

提交贡献即表示你同意：你的贡献可以由 LifeBook 书坊用于 LifeBook 产品与服务。

LifeBook 书坊负责本项目的组织、编辑、发布、质量控制、授权管理和贡献者回馈安排。

贡献者保留其贡献的署名权益。具体署名方式、贡献记录方式和回馈方式，由 LifeBook 书坊根据贡献类型、贡献量、合并情况、项目阶段和实际运营情况制定和调整。

LifeBook 书坊可以根据项目规则和实际情况，向重要贡献者提供署名、项目权益、优惠、奖励、分成或其他形式的回馈。具体比例、条件和执行方式由 LifeBook 书坊自主决定。

### 第三方商业使用

第三方商业使用本项目译文、EPUB、整理内容或其他非代码创作内容，必须另行取得 LifeBook 书坊及相关权利人的授权。公开许可证不自动授予第三方商业使用权。

### 代码贡献

提交代码贡献即表示你同意该代码可以按 [LICENSE.md](LICENSE.md) 中的 MIT License 发布，除非该文件或目录明确说明使用其他许可证。

### 不要提交的内容

请不要提交：

- 现代受版权保护的译本内容。
- 从盗版站、网盘或来源不明 EPUB 中复制的文本。
- 你没有权利提交的封面、图片、字体、注释或其他素材。
- 会泄露隐私、账号、密钥或付费资源的内容。
