# 天文学领域规则 / Astronomy Rules

## 高风险对象

- 黄道、赤道、天球、地球、太阳、月亮、行星、恒星、星座。
- 本轮、均轮、偏心圆、等分点、黄经、赤纬、纬度、经度。
- 星表、弦表、历法、观测值、角度和时间单位。

## 硬规则

- 天文模型术语必须先锁定，再翻译相关章节。
- 本轮、均轮、偏心圆、等分点等模型术语必须进入 `qa/technical/astronomical_model_registry.csv`。
- 星名、星座名、行星名和天体名应优先查询权威天文学名词来源。
- 角度和六十进制数值必须进入数值校验记录。
- 模型图必须核对圆心、半径、标签、方向、点名和正文引用。
- 古代几何天文学模型必须按原文呈现；现代解释只能放入注释或导读。

## P0 错误

- 把古代地心模型术语误改成现代天体物理术语。
- 星表或角度数值漂移。
- 图中点名、圆名和正文说明不一致。
