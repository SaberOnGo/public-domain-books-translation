# 古典科学技术出版控制 / Technical Publication Control

## 控制目标 / Control Goals

古典科学、数学、天文学、地理学、光学、力学、医学等技术性公版书，必须把语言翻译、文本校勘、术语控制、数值校验、图表校验和 EPUB 生产视为同一个系统。

## 必须分离的层级 / Required Separation

- 原文层：底本、witness、异文、残损、校勘。
- 语言层：源语言语法、句法、专名、转写、译注。
- 技术层：定义、命题、证明、模型、单位、数值、图表。
- 出版层：EPUB nav、XHTML、图表格式、可访问性、metadata。

## P0 错误 / P0 Errors

- 使用版权不清的现代译本作为底稿。
- 关键术语未锁定就批量翻译。
- 数学证明链被译断。
- 图表标签和正文引用不一致。
- 表格行列或数值发生漂移。
- 现代解释混入正文冒充原文。
- 复杂图表没有可访问说明。

## P1 错误 / P1 Errors

- 局部术语漂移。
- 图表重绘源文件缺失。
- 单位换算未说明精度。
- 参考译本差异未记录。
- 单章审计缺少证据位置。
