# 042_glossary.md 信达雅复核

# 结论

本章是术语表，**问题主要不在行文而在术语对位**：有多处把船体部件、船名/船型、海战术语、航海动作译错或译偏，且部分条目出现**中文定义与英文原义反向**的硬伤。  
整体判断：**信、史风险偏高，版面层面的重复与条目错位也较多**。  
静态扫描未报硬性问题，但就术语表本身看，**仍有若干必须修复项**，否则会直接误导正文阅读。

# 信

## P0

1. **“艉锚（即大艉锚和小艉锚）。船首所携带的锚。”**
   - 英文：`Bower anchors ... The anchors carried at the bows of a vessel.`
   - 问题：**锚名和位置对不上**。`bower` 是船首锚，不是“艉锚”；中文把“bow”译反了。
   - 建议改法：  
     - `艉锚` → **`船首锚`** 或 **`首锚`**  
     - 后句改为：**“船首所携带的锚。”**保留即可

2. **“外舷锚（= shoot anchor）”**
   - 英文原文是 `sheet anchor (= shoot anchor)`，属于旧拼写/误印体例，核心义是**大副锚/压舱主锚**，不是“外舷锚”。
   - 问题：术语本体错位，且中文未体现“备用大锚/应急大锚”的功能。
   - 建议改法：  
     - `外舷锚` → **`大副锚`** 或 **`应急大锚`**  
     - 注释可改为：**“遇重大危险时投放或放下的锚……”**

3. **“前帆。由桅杆借助斜桁撑出的一种四角形帆……”**
   - 英文：`Sprit-sail. A quadrangular sail stretched from the mast by the help ... of a sprit...`
   - 问题：`sprit-sail` 是**斜桁帆/斜撑帆**，不是泛泛“前帆”。“前帆”在中文里会被理解为 fore sail 或船首帆。
   - 建议改法：  
     - `前帆` → **`斜桁帆`**  
     - 释义保留，必要时把“纵帆”对比句补顺。

4. **“顶风漂泊（lie to）。”**
   - 英文：`Lay to (lie to). To reduce sail to the lowest limits, so as to become nearly stationary.`
   - 问题：`lie to` 不是“漂泊”，而是**顶风停船/收帆停航**；“漂泊”会把动作链误导成随风漂流。
   - 建议改法：  
     - `顶风漂泊` → **`顶风停船`** 或 **`收帆待风`**

5. **“大班。受船主雇用随船出航并监督货物的人。”**
   - 英文：`Supercargo. A person employed by the owners of a ship to go a voyage and to oversee the cargo.`
   - 问题：`supercargo` 不是泛称“大班”，而是**大班/押货员/船上货务监督**；如不加限定，读者会把它混成近代商业“大班”。
   - 建议改法：  
     - `大班` → **`押货员`** 或 **`货务监督（大班）`**

6. **“前帆桁。sprit 的另一名称。”**
   - 英文：`Sprit-sail yard. Another name for the sprit.`
   - 问题：这里把 `sprit-sail yard` 译成“前帆桁”会和上一条“前帆”互相打架，而且 **sprit 是斜桁/支帆杆**，不是“前帆桁”这种常规前桅横桁。
   - 建议改法：  
     - `前帆桁` → **`斜桁`** 或 **`支帆杆`**

## P1

1. **“横列（枪手队列）。”**
   - 英文：`File (of musketeers)...`
   - 问题：`file` 是**纵列/一列**，不是“横列”。中文解释也写成“一个紧接一个站成的队列”，与“横列”自相矛盾。
   - 建议改法：  
     - `横列` → **`纵列`**  
     - `枪手队列` 可改为 **`火枪手纵列`**

2. **“清洁。清洁的船底没有藤壶和海草，因此航行较快。”**
   - 英文：`Clean. A clean ship is one whose bottom is free from barnacles and weed...`
   - 问题：定义基本对，但“清洁”太泛，缺少航海术语感；不过可接受。
   - 建议改法（可选）：  
     - `清洁` → **`船底洁净`** 或 **`洁净船底`**

3. **“战列舰。指船体和火力足以在战列线上占一席之地的军舰。”**
   - 英文：`Ship of the line...`
   - 问题：可读，但中文术语更常用 **“列线舰”/“战列舰”**；当前与条目 `Line` 的互引还算通顺。
   - 风险：历史语境里“战列舰”容易被现代读者联想到19世纪铁甲舰，但这里尚可接受。
   - 建议：如全书统一现代常用译法，可保留；若强调18世纪，可考虑 **“列线舰（战列舰）”**。

4. **“左舷（或港舷）。” / 后文又有“左舷（或左舷）。”**
   - 英文：`Larboard (or port)` / `Port (or larboard)`
   - 问题：第二处括注明显重复，属于版面/编辑错误。
   - 建议改法：  
     - 第二处改为 **“右舷（或 starboard）”** 对应项不在此处；若保留当前条目，则至少删去重复的“或左舷”。

5. **“收帆。把帆卷起。”**
   - 英文：`Hand (the sails). To furl.`
   - 问题：`hand` 更准确是**收起/卷起帆**，但术语“收帆”可以；只是和前文 `lay to` 等动作容易混。
   - 风险：低，暂可接受。

6. **“风干肉。”**
   - 英文：`Jerk. To cure meat...`
   - 问题：`jerk` 是**腌制风干肉/晒干肉**，译成名词“风干肉”可以，但动词义丢失。
   - 建议改法：  
     - `风干肉` → **`腌晒肉`** 或 **`风干（肉）`**

## P2

1. **“可可豆。巧克力果仁。”**
   - 英文：`Cacao. Chocolate nuts.`
   - 问题：可接受，但“巧克力果仁”略现代化。
   - 建议：若求历史感，可改 **“可可豆”** 并删去解释性“巧克力果仁”。

2. **“小舢舨。” / “小型马来船。”**
   - 英文：`Pinnace. ...` / `Proa. A small Malay vessel.`
   - 问题：`pinnace` 译“小舢舨”偏口语；`proa` 译“小型马来船”可但不够术语化。
   - 风险：低，但会影响术语统一性。

# 达

1. **“前桅、前顶桅、前上桅、前天空桅……”**
   - 英文：`topgallant-mast, royal mast`
   - 问题：`sky-sail` 类似层级被译成“天空桅”虽可懂，但明显现代直译腔，缺少旧海军术语惯例。
   - 建议：  
     - `前天空桅` → **`前天顶桅`** 或统一为 **`前上桅杆/前天帆桅`**（需全书一致）

2. **“拖挪。借助缆索将船只移到另一位置……”**
   - 英文：`Warp. To move a vessel into another position by hauling upon a hawser...`
   - 问题：准确但偏说明书腔，未体现航海语气。
   - 风险：作为术语表尚可，不算硬伤。

3. **“转向。将船转过来，先让船顺风前进，然后转到新的航向上。”**
   - 英文：`Wear (a ship)...`
   - 问题：`wear` 是**顺风掉头转向**，中文“转向”太泛，读者难以与 `tack` 区分。
   - 建议改法：  
     - `转向` → **`顺风转向`** 或 **`掉顺风头`**

4. **“迎风……到达某物的上风处。”**
   - 英文：`Weather`
   - 问题：术语区分不错，但“到达某物的上风处”略拗口。
   - 风险：低。

# 雅

1. **“船体和火力足以在战列线上占一席之地的军舰。”**
   - 问题：这是可用的说明文，但略显现代军事说明腔；若整书正文也如此，会削弱18世纪航海纪实的节制感。
   - 建议：术语表可保留，正文中尽量避免再重复这类解释句式。

2. **“翻船修整。将船身一侧放倒，使另一侧露出以便清洗的作业。”**
   - 英文：`Careening...`
   - 问题：准确，但“作业”较工程化。
   - 建议：可微调为 **“放倒一舷以清洗船底的操作”**，更自然也更克制。

3. **“缠护。用旧三英寸绳索螺旋状地包缠缆索，以防磨损。”**
   - 问题：术语表风格偏硬，尚可；但“缠护”不是常见中文航海术语，陌生感较强。
   - 风险：中等，若正文频繁出现，读者会停顿。

# 史

1. **“艉锚 / 船首锚”反置**
   - 这是最严重的史实/器物位置错误之一。锚的位置一旦错，后文涉及起锚、下锚、应急锚使用的理解都会偏。

2. **“外舷锚”替代 `sheet anchor`**
   - `sheet anchor` 是帆船时代的重要应急锚术语，误译会影响对风暴、抢险、弃船等情节的理解。

3. **“顶风漂泊（lie to）”**
   - `lie to` 是典型海战/风暴术语，涉及舰船在暴风中控船方式。译成“漂泊”会把**控制状态**误成**失控状态**。

4. **“前帆”代替 `sprit-sail`**
   - 属于帆装史知识点错位；对18世纪帆装构型的理解会受影响。

5. **“横列（枪手队列）”**
   - 兵法/队列概念反了。若正文涉及登陆、操炮或卫队阵列，会连带误读战术动作。

6. **“大班”**
   - 在广州、澳门语境里，`supercargo` 若不处理好，会与中英贸易史中“商馆大班/洋行大班”的后世用法混淆。术语表最好给出更中性的“押货员/货务监督”。

# 版

1. **“左舷（或左舷）。”**
   - 明显重复，属于版面/校对层面问题。
   - 建议修正对应条目，避免术语表自相矛盾。

2. **“八里硬币。见‘八里硬币’。”**
   - 交叉引用存在**自指**，疑为条目命名未统一或回链错误。
   - 建议改为：  
     - 若对应 `Eight`：**“八。见‘八里硬币’。”**  
     - 或将条目名统一为 **“八分银币”/“西班牙八里硬币”**，再做回链

3. **“轻便艇”出现两次，分别对应 `Cutter` 与 `Cutter. See Boats.`**
   - 这本身不是错，但若术语表中条目顺序与回链不一致，容易造成查阅混乱。
   - 建议人工检查重复条目是否有遗漏译注。

4. **“前桅、前顶桅、前上桅、前天空桅……”**
   - 术语层级中文化不统一，容易和正文中的其他译名冲突。
   - 建议全书统一一套桅杆命名表。

# 必须修复

1. **“艉锚（即大艉锚和小艉锚）。船首所携带的锚。”**  
   → 改为 **“船首锚（即大船首锚和小船首锚）。船首所携带的锚。”**

2. **“外舷锚（= shoot anchor）”**  
   → 改为 **“大副锚（= sheet anchor）”** 或 **“应急大锚”**

   → 改为 **“顶风停船（lie to）。”**

4. **“前帆。由桅杆借助斜桁撑出的一种四角形帆……”**  
   → 改为 **“斜桁帆”**

   → 改为 **“纵列（火枪手纵列）。”**

   → 改为 **“斜桁。sprit 的另一名称。”**

# 建议人工复核

1. **`ship of the line` 是否统一译为“战列舰”还是“列线舰”**  
   - 目前可用，但建议全书统一，避免现代军舰概念干扰。

2. **`supercargo` 的译名策略**  
   - 建议在术语表定为 **“押货员（大班）”**，并与广州/澳门商馆语境核对。

3. **`sprit-sail`, `sprit-sail yard`, `foreyard`, `fore-cap` 这一组桅帆件术语**  
   - 当前多处译名相近、层级易混，建议由懂帆装史的人工统一。

4. **`sky-sail mast` 类“前天空桅/主天空桅/后天空桅”译法**  
   - 需确认全书是否采用同一套中文帆装层级，不要正文与术语表互相打架。

5. **`Canton River`、`Macao`、`Viceroy`、`mandarin` 在正文中的对照一致性**  
   - 本章虽是术语表，但与广州/澳门章节强相关，建议核查正文是否完全按既定地名与官职译法执行。

6. **`lay to / wear / warp / fetch / fore-reach / overreach` 等动作链术语**  
   - 本章是航海动作密集项，建议人工抽查正文中这些词是否与术语表一致，防止前后章术语漂移。
