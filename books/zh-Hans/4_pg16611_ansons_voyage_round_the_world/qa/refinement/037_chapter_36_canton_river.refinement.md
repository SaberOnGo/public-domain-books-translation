# 037_chapter_36_canton_river.md 信达雅复核

# 结论

本章**无明显 P0/P1 硬伤**，整体对原文的事实链、时间顺序、外交交涉逻辑和海上行动线保持得比较稳。  
但有若干 **P2 级措辞与术语风险**：个别官场/航海术语略有现代化或抽象化，少数句子把原文的精确性压成了“说明腔”。如果不修，读者大体能懂，但会损失 18 世纪航海纪实的质感与历史语域准确度。

# 信

- **时间、数字、地点、动作链总体准确。**  
  7月14日、16日、20日、28日的事件推进无错位；虎门、第二道沙洲、澳门、广州、北京朝廷的空间关系也基本对。
- **战利/俘获关系未误伤。**  
  `prize` 在这里译作“捕获船/战利船”都能成立，没有错成“奖品”。
- **外交叙事主干完整。**  
  “询问—申报武装—税费争执—释放俘虏—总督回信—送俘虏去澳门—停泊等季风”这一整条链条保留完整。

**剩余风险：**  
个别地方的用词把原文的海军与清代官场语气磨平了，尤其是“税”“命令”“拜访”“当政者”等词，略显现代说明文风。

# 达

- **可读性合格，句法基本顺。**  
  中文拆句较自然，没有把长句硬译成一整坨。
- **信息传递基本到位。**  
  读者能看懂：百夫长号进入广州河、英国军舰身份申报、官员查问火炮与弹药、广州方面要求税费与放人、最后双方各退一步。

**P2 风险：**
1. **“皇税”不够贴合原文语境。**  
   原文是 *the Emperor's duty*，指港口税/关税一类，不是泛指“皇税”这种现代抽象词。
2. **“当政者”过泛。**  
   原文 *the Regency* 指广州方面的官府/执政机构，不宜抽象成“当政者”，容易丢失官场层级。
3. **“与其臣民进行大规模贸易的人”略生硬。**  
   原文 *carried on a great commerce with his subjects* 是“与其臣民做大买卖”，这里现在译法有点书面化、解释化。

# 雅

- **整体克制，没过度文学化。** 这是优点。
- 但有几处略显“说明腔”：
  - “**看他会多么准确地遵守自己的命令**”  
    原文是强调“按指示办事”的可证实性，不是自我命令的姿态；中文略拗。
  - “**由于召集举行此礼所必需的官员和士兵**”  
    过于公文化，原文只是“召集必要的官员和士兵”，不必抬高成仪式性套话。
  - “**命司帐官为他们备好八日口粮**”  
    “司帐官”略带现代职务说明感；原文是 *purser*，更像“军需官/司粮官”。

**P2 建议：**
- “**由他们亲自作证，看他会多么准确地遵守自己的命令**”  
  可改为：**“让他们亲眼作证，看他如何严格遵照指示行事。”**
- “**命司帐官**”  
  可改为：**“命军需官”** 或 **“命管粮官”**。

# 史

- **历史语境基本守住。**
  - 广州/澳门/总督/官员/税费/领航员/季风，这些关键历史语汇都在。
  - “英王所属的战船”“西班牙国王”“皇帝”对位清楚。
- **但有两处历史语域不够精准：**
  1. **“皇税”**  
     这会把港口税泛化成“皇帝征税”的现代感表达，容易偏离 18 世纪对港务税、过港税的具体行政概念。
  2. **“拜访总督”**  
     原文是礼节性“拜会/觐见”而不是一般社交“拜访”。在广州官场语境里，“觐见/谒见”更稳。

**P2 建议：**
- “**拜访总督**”  
  可改为：**“谒见总督”** 或 **“拜谒总督”**。
- “**皇税**”  
  可改为：**“皇帝的税”**（若保留原味）或 **“港税/关税”**（若按语境落地）。

# 必须修复

本章**无 P0/P1 必修硬伤**。  
以下项目建议按 P2 处理，但若全书统一术语，最好同步微调：

1. **“皇税”**
   - 位置：`那官员听说安森先生竟可免缴一切税费...`
   - 问题：过度现代抽象，弱化港口税语境
   - 建议改法：  
     - **“天子之税”**（保留古感）  
     - 或 **“港税/关税”**（按语境落地）

2. **“拜访总督”**
   - 位置：`打算前往广州，拜访总督`
   - 问题：礼节层级偏弱
   - 建议改法：  
     - **“谒见总督”**  
     - 或 **“拜谒总督”**

3. **“当政者”**
   - 位置：`恐怕会过分惊动当政者`
   - 问题：官府主体不够明确
   - 建议改法：  
     - **“恐怕会过分惊动官府”**  
     - 或 **“恐怕会惊动广州当局”**

4. **“准将的命令”**
   - 位置：`看他会多么准确地遵守自己的命令`
   - 问题：原文是 instructions，不是“自己的命令”
   - 建议改法：  
     - **“严格遵照他的指示”**

# 建议人工复核

以下为**P2 级**，建议人工再看一遍术语统一性：

1. **“捕获船 / 战利船”**
   - 本章两种译法都可，但建议全书统一优先用 **“捕获船”**，更贴近本书叙事中的 *prize*。
   - 现文：`率其捕获船...`、`另一艘是他缴获的战利船`
   - 建议：统一成 **“捕获船”** 或 **“战利品船”**，避免同章摇摆。

2. **“司帐官”**
   - 位置：`命司帐官为他们备好八日口粮`
   - 风险：现代行政口吻偏重
   - 建议：核对全书对 *purser* 的统一译法，优先考虑 **“军需官”**。

3. **“火枪”**
   - 位置：`四百支火枪`
   - 评估：可接受，但若全书已约定海战武器更偏“火铳/燧发枪”，此处需统一。
   - 原文：*four hundred fire locks*，更接近燧发枪一类，不必细化也可保留“火枪”，但要统一。

4. **“第二道沙洲”**
   - 位置：`带过第二道沙洲`
   - 风险：中文可懂，但需确认全书对 *second bar* 是否固定译为“第二道沙洲”而非“第二道浅滩/沙坝”。
   - 这不影响理解，但关系到航海术语一致性。
