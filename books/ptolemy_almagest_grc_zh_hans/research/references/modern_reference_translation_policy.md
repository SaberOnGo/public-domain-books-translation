# 现代参考译本政策 / Modern Reference Translation Policy

policy_status: `PASS_FOR_RESEARCH_ONLY`

## Toomer 英译本

G. J. Toomer 的现代英译本可作为权威 reference witness，但版权仍需按现代出版物处理。

允许：

- 校对术语理解。
- 定位疑难段。
- 比较数学/天文学解释。
- 查找现代学术注释方向。

禁止：

- 直接转译。
- 复制英文表达。
- 复制注释。
- 复制图表或表格。
- 让英文译本结构替代古希腊文底本结构。
