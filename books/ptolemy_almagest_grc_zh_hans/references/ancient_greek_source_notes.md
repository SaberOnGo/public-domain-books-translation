# 古希腊文源语言说明 / Ancient Greek Source Notes

本文件只记录古希腊文进入简体中文时容易出现的源语言、底本和校勘风险。更通用的简体中文译文质量规则见 `template/epub_pipeline/targets/zh-Hans/quality_framework/`。

This file records Ancient Greek source-language and edition risks when translating into Simplified Chinese.

## 底本与版本 / Edition

- 必须记录所用古希腊文底本、编辑者、出版年份、卷册、页码/行号或章节编号。
- 扫描件、OCR、TEI/XML、网页转写必须区分。未校 OCR 不得当作可靠原文。
- 若同一作品有多个校勘版，必须说明选用理由。
- 若采用公版校勘版，仍要记录该校勘版本身的来源和公版证据。

## 古希腊文干扰 / Greek Interference

- 不要把希腊文长周期句硬搬成中文长句；应拆分，但保留因果、转折、递进和论证关系。
- 分词结构常承载时间、原因、条件、让步或伴随动作，不能机械省略。
- 冠词、代词、指示词和省略主语经常依赖上下文，中文必须补足可理解关系，但不得补出原文没有的事实。
- 被动、无人称表达和抽象名词结构应转成自然中文，同时保留论证重心。
- 引语、转述和作者评论要分清；不得把作者自己的判断误译成引用者观点。

## 专名与术语 / Names and Terms

- 人名、地名、神名、民族名、学派名必须建立统一译名策略。
- 可使用通行中文译名时，应记录来源；没有通行译名时，建立转写规则。
- 技术术语应区分古典概念、现代等价概念和译者说明，不能把现代术语硬塞回古代文本。

## 参考译本边界 / Reference-Witness Boundary

- 第二语言译本只能帮助理解和校对，不得成为中文译文底稿。
- 参考译本和希腊文底本不一致时，必须回到希腊文判断，并记录差异。
- 仍受版权保护的参考译本不得复制其措辞、注释、图表或表格。
