# 《金属巨兽》文体画像 / Style Profile

## 基本判断

*The Metal Monster* 是 1920 年代美国通俗杂志传统中的奇幻/科幻探险小说。叙述核心不是硬科幻解释，而是由探险、异域山地、无机生命、金属几何体和准神话式女王形象组成的“惊异感”。

中文译文目标：

- 忠实保留第一人称见闻录口吻。
- 长句可拆，但不得削弱递进、排比和宏大修辞。
- 科学术语、植物学、地理、历史名词要准确，奇观描写要有画面。
- 对 1920 年代源文中的东方主义、种族/民族称谓和殖民时代语气，不替作者粉饰，必要时在译注或 QA 中记录处理理由。

## 叙述声音

- Goodwin 的声音：受过高等教育、观察细密、带科学训练，同时极易被奇观震撼。
- Merritt 式铺陈：形容词密集、光色与金属质感强、常用排比和节奏递进。
- 对话：人物阶层和情绪要区分，避免所有人都说成现代口语。

## 中文策略

- 章节标题：目录短，页面保留副标题。
- 人名：音译统一，首次自然出现可在术语表记录英文原名。
- 科学/金属意象：避免偷懒译成“东西”“物体”，按上下文区分形体、块体、群体、金属生命、金属之城。
- 高潮动作：长段落按动作阶段拆分，保持阅读推进。
- 标点：中文引号、中文破折号和全角标点，不得留下英文半角标点污染。

## 高风险问题

- `metal` 不应一律译作“金属物”，有时是物质，有时是生命群体，有时是文明。
- `shape/forms` 是本书核心意象，应结合几何和群体生命语境处理。
- `Norhala` 既是人名又具有神话/女王气质，译文不应把她写成普通人物。
- 西藏、波斯、印度、古代民族称谓需谨慎，不用现代政治解释覆盖源文语境。
