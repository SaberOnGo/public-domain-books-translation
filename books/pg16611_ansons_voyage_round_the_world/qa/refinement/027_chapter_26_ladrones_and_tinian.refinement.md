# 027_chapter_26_ladrones_and_tinian.md 信达雅复核

# 结论

本章总体**可读，主线与关键信息基本对齐英文原文**，没有发现 P0 级硬伤。  
但存在若干 **P1/P2 级局部问题**，主要集中在：

- 个别词语选择略显机械或现代化；
- 船只名物的表达不够准确/不够自然；
- 末段有一处明显的中文病句感，建议修复；
- 另有少量细节可进一步压实，以避免术语感偏弱。

---

# 信

## P1
1. **“去控制那艘小船”**  
   - 原文：`we therefore immediately despatched the pinnace to secure the bark`
   - 问题：`secure` 在此是“扣住、控制、看守住”，而不是一般口语里的“去控制”。“控制”偏现代行政/安保语感，略硬。
   - 建议改法：  
     - `我们立刻派出舢板，去扣住那艘小船`  
     - 或 `……去看住那艘小船`
   - 定位：`我们立刻派出舢板，去控制那艘小船`

2. **“那也是岛上唯一的一件船只”**
   - 原文：`the only embarkation on the place`
   - 问题：`一件船只` 是明显的中文搭配失真，机械感强，破坏文体。
   - 建议改法：  
     - `……军士告诉我们，这也是岛上唯一的一艘船`
     - 或 `……岛上也只有这一艘船`
   - 定位：`军士告诉我们，这也是岛上唯一的一件船只`

## P2
3. **“帕罗阿船”**
   - 原文：`a proa`
   - 问题：虽能对应原词，但在中文里“帕罗阿船”不算通行译法，读者可能不知所指；且本书后文若多次出现，术语一致性会受影响。
   - 建议改法：  
     - `一艘独木舟（proa）`  
     - 或 `一艘帕罗阿独木舟`
   - 定位：`看见天宁岛与阿圭让岛之间南面有一艘帕罗阿船扬帆而行`

4. **“后甲板的炮”**
   - 原文：`upper and quarter-deck guns`
   - 问题：`quarter-deck` 译作“后甲板”大体可通，但在航海语境里略泛，容易丢失甲板层级感。  
   - 建议改法：  
     - `上层甲板和后甲板上的炮`
   - 定位：`并在上层甲板和后甲板的炮上装填霰弹`

# 达

- 叙述顺序基本忠实：发现岛屿 → 望锚地失败 → 继续南趋 → 发现天宁岛 → 伪装马尼拉大帆船 → 俘获帕罗阿船 → 获知补给信息 → 下锚。
- 长句拆分总体顺畅，因果链没有断裂。
- 但个别句子有“说明书化”倾向，尤其是末段“去控制那艘小船——军士告诉我们……”这一句，语势偏硬，建议顺手润色。

# 雅

## P1
1. **“去控制那艘小船”**
   - 这类表述偏现代管理/安保腔，不够 18 世纪航海纪实的节制感。
   - 建议改为更直接的动作词：`扣住 / 看住 / 夺下 / 控住`。

2. **“那也是岛上唯一的一件船只”**
   - 属于明显的机械翻译腔，已影响全句质地，必须修。

## P2
3. **“令人愉悦的景象”**
   - 作为小标题基本可用，但略带现代散文化色彩。  
   - 若想更贴近原文 `PLEASING SCENES` 的节制，可考虑：
     - `可喜的景象`
     - `愉悦所见`
   - 但这不是硬伤，可保留。

- **“阿纳塔坎 / 塞里甘 / 帕萨罗斯 / 塞班 / 天宁 / 阿圭让”** 与原文对应基本正确，未见明显岛名错配。
- **“关岛总督”** 对应 `Governor of Guam` 正确。
- **“马尼拉大帆船”** 对应 `Manila galleon` 正确。
- **“二十二寻水深”** 对应 `twenty-two fathoms` 正确。
- **“1742年8月26日”** 与原文时间一致。
- 旧时代偏见在本章主要不突出；“西班牙人”“印第安人”等历史称谓均按原文保留，未见扩大化改写。

# 版

- 章节标题结构清楚，层级可读。
- 小标题 `## 令人愉悦的景象` 可以接受，但如全书各章标题风格需统一，可考虑与其他章节的标题粒度保持一致。
- 末段的 `一件船只` 属于明确版面终稿问题，建议在出 EPUB/纸书前修掉。
- `帕罗阿船` 是否纳入术语表，建议统一处理，否则后文重复出现时可能不稳。

# 必须修复

1. **`去控制那艘小船`**
   - 改为：`去扣住那艘小船` / `去看住那艘小船`

2. **`岛上唯一的一件船只`**
   - 改为：`岛上唯一的一艘船`
   - 这是明显中文病句，必须修。

# 建议人工复核

1. **`帕罗阿船` 的术语策略**
   - 是否统一译作 `独木舟`、`帕罗阿独木舟`，还是保留音译。
   - 若全书多处出现，建议术语表固定。

2. **`后甲板` 对应 `quarter-deck` 的一致性**
   - 本章可通，但建议检查全书是否统一译法，避免一会儿“后甲板”，一会儿“艉楼/尾甲板”。

3. **小标题 `令人愉悦的景象`**
   - 非硬伤，但若全书标题偏纪实，可考虑是否需要更克制的题名。

**总体评价：本章达标，但有 1 处明确病句、1 处动词腔调偏现代，建议在终版前修正。**
